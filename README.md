# mv_attachment

PrestaShop module for attaching files to outgoing e-mails based on template and language.

## English

### Compatibility

- PrestaShop 8.x, 9.x
- Module: `mv_attachment`
- Current version: `1.1.0`

### Installation

1. Copy the module into `modules/mv_attachment`.
2. In PrestaShop back office, open **Modules > Module Manager**.
3. Install the module **E-mail Attachments**.

### Features

- Attachment management in module configuration
- Assigning attachments to specific e-mail templates
- Language-specific attachment assignment
- Supported formats: PDF, ZIP, RAR, TXT, DOC, DOCX, XLS, XLSX, ODT, ODS, PNG, JPG, GIF

### Project Structure

- `mv_attachment.php` - main module file
- `config.xml` - module metadata
- `translations/` - translations
- `upgrade/` - upgrade scripts
- `views/` - templates and assets
- `controllers/` - module controllers

## Slovensky

PrestaShop modul pre dynamicke pripajanie priloh k e-mailom podla sablony a jazyka.

### Kompatibilita

- PrestaShop 8.x, 9.x
- Modul: `mv_attachment`
- Aktualna verzia: `1.1.0`

### Instalacia

1. Skopirujte modul do priecinka `modules/mv_attachment`.
2. V administracii PrestaShop otvorte **Modules > Module Manager**.
3. Nainstalujte modul **E-mail Attachments**.

### Funkcie

- Sprava priloh v administracii modulu
- Priradenie prilohy ku konkretnej e-mail sablone
- Priradenie prilohy podla jazyka
- Podpora viacerych formatov (PDF, ZIP, RAR, TXT, DOC, DOCX, XLS, XLSX, ODT, ODS, PNG, JPG, GIF)

### Struktura projektu

- `mv_attachment.php` - hlavny subor modulu
- `config.xml` - metadata modulu
- `translations/` - preklady
- `upgrade/` - upgrade skripty
- `views/` - sablony a assets
- `controllers/` - kontrolery modulu
